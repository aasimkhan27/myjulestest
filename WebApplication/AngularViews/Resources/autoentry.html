How to use the Mobile App of Expense Claim?

  • Request your Manager to setup the user (GM should contact wenodo to setup the user for expense claim)
  • Download the app and login using your user id and the password
  • Use the Expense report from the dropdown arrow “1”
  • Go to Reports
  • Create a New Expense Report
  • Add Report Name (Expense Claim Followed by your Name)
  • Report Date (Use weekending Sunday date for which your are submitting the claim.
  • Add a note if required (For example if only for a specific reason)
  • After you create the Report go back to the main Expense Screen and taking the pictures of your receipts.
  • Fill all the fields before assign the expense to the report
  • Once you take all the pictures and filed all the fields of your claims go back to the following screen and assign your bills to the respective weekending claims.
  • Once you ready you can press the green “Submit Report” for approval.

For any queries, kindly email us at support@wenodo.com
