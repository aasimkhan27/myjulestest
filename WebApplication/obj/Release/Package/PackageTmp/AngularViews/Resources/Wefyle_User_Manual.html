Log-in

How to log-in to the cash-up app?

1. Launch app.wenodo.com & Log-in with your credentials.

2. From the App-Suite, click on Wefyle App.

3. Click on Upload File from the left.

  • Restaurant- select the restaurant from the dropdown.
  • Branch – select the branch.
  • Upload Type – select the type of file you like to upload.
  • General – select the general file and click upload icon.
  • Approval Required – click on the approval box if it needs an approval.
  • Comment – Leave any comment you feel relevant.
  • Click on SAVE to send the process or CANCEL to start over
  • Payroll – If it’s a payroll file please select the month/frequency/payroll end date, upload the file and click the upload icon.
  • Approval Required – approval for payroll file is selected by default.
  • Comment – Leave any comment you feel relevant.
  • Click on SAVE to send the process or CANCEL to start over.

4. Click on the Upload List from the left :

Here you can see the complete list of uploads with its approval status.

  • Restaurant – select the restaurant from the dropdown.
  • Branch – select the branch from the dropdown.
  • Month -select the month you like to filer from.
  • Result – once selected you fill the relevant uploads based on applied filters.

5. Approval

Here you can filter to check the status on your uploaded files.

  • Restaurant – select the restaurant from the dropdown.
  • Branch – select the branch from the dropdown.
  • Month -select the month you like to filer from.

For Approver

1. To Approve

  • Click on the action sign on the right side of PENDING APPROVAL
  • On clicking a new screen will appear.
  • You can download any files from the list.
  • Save then to your local drive, make amendments if necessary.
  • Reject or Approve with comment.

2. Amend and Approve

  • You can also make amendments to the downloaded file
  • And upload again with comments.
  • Refer above steps for uploading a file.

3. Final Action

  • Once you click to approve the status will change to approve.
  • If rejected the uploader will receive a rejection notification.
  • If you will wish you upload a file, click upload and select the file.
  • You can also leave comments in the comments section.
